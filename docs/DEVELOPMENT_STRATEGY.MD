# Development Strategy 🧠

## 1. AI-First Workflow
Every line of code is developed using **GitHub Copilot Business**. We leverage AI not just for speed, but for architectural optimization.

## 2. Compulsory Commenting Policy
In line with our educational mission, **every single line of code must be commented**. Comments must explain:
* **What** the line does.
* **Why** it is necessary for the logic of the program.

## 3. Iterative MVP Focus
We prioritize core functionality over "perfect" features. We follow the principle of "Build, Test, Learn" to ensure rapid delivery of educational value.

## 4. Hardware-Accelerated Dev
Utilization of the local **RTX 5070** and **AMD 7900X3D** for local LLM testing and high-speed compilation before cloud deployment.
