# Contributing to TechTutorPlay Lab 🎓

First off, thank you for considering contributing to TechTutorPlay Lab! It's people like you that make this educational project a great tool for students.

## 🚀 How Can I Contribute?

### Reporting Bugs
* Check the [Issues tab](https://github.com/TechTutorPlay-Lab/TechTutorPlay-Lab-Core/issues) to see if the bug has already been reported.
* If not, open a new issue. Clearly describe the bug and include steps to reproduce it.

### Suggesting Enhancements
* Open a new issue with the tag "enhancement".
* Describe the feature you'd like to see and why it would be useful for the Lab's educational goals.

### Code Contributions
1. Fork the repository.
2. Create a new branch for your feature (`git checkout -b feature/AmazingFeature`).
3. Commit your changes (`git commit -m 'Add some AmazingFeature'`). 
   * **Note:** Please include comments in your code explaining the logic.
4. Push to the branch (`git push origin feature/AmazingFeature`).
5. Open a Pull Request.

## 🛠 Tech Standards
* **Languages:** We primarily use C# (.NET 8/9), Python, and PowerShell.
* **Environment:** Ensure compatibility with Windows 11 Pro and Windows Server 2025.
* **Style:** Maintain clean code and professional documentation.
