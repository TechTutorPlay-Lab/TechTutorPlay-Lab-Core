# TechTutorPlay Lab - Infrastructure Specs

## Local Development Environment
* **Workstation CPU:** AMD Ryzen 9 7900X3D (12-Core, Zen 4 V-Cache)
* **Graphics/AI Accel:** NVIDIA GeForce RTX 5070 (12GB VRAM Ready)
* **Memory/Storage:** 2TB WD Blue High-Performance Storage
* **Visuals:** Samsung Odyssey G4 @ 240Hz for real-time monitoring

## Security & Reliability
* **Authentication:** Google Advanced Protection Program enabled.
* **Hardware Security:** Thetis FIDO2 Security Keys for all administrative access.
* **Version Control:** Managed via GitKraken on localized WD storage.

## Project Goal
To leverage local hardware for AI model prototyping and scale to **Microsoft Azure/Google Cloud** for global deployment.
